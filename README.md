# PentaQ
PentaQ – Visual quantum circuit composer for education. Build circuits with up to 5 qubits, drag-and-drop gates (X, H, CNOT, Rx, Ry, Rz, etc.), simulate with Qiskit, and visualize results via interactive histograms and 3D Bloch spheres. Built with Next.js, TypeScript, Three.js, and Tailwind CSS. Perfect for students and teachers.
